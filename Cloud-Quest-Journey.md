# AWS Cloud Quest Journey

## About

As part of my AWS Cloud Support Engineer learning journey, I regularly practice using **AWS Cloud Quest: Cloud Practitioner**.

This game provides hands-on AWS scenarios where I solve real cloud-related tasks in an interactive environment.

## My Goal

My objective is not only to complete the game but to strengthen my practical AWS skills by:

* Understanding AWS services through practice.
* Solving real-world cloud challenges.
* Improving troubleshooting skills.
* Reinforcing concepts learned from AWS documentation.
* Developing the mindset of a Cloud Support Engineer.

## Daily Learning

Each day, I complete new Cloud Quest challenges and document what I learn.

For every session, I focus on:

* The AWS services involved.
* The objective of the challenge.
* Problems I encountered.
* How I investigated the issue.
* The solution I applied.
* Key lessons learned.

## Why I Use Cloud Quest

Cloud Quest allows me to learn by doing instead of only reading documentation.

It helps me:

* Practice AWS concepts in realistic scenarios.
* Build confidence using AWS services.
* Improve problem-solving skills.
* Think like a Cloud Support Engineer when troubleshooting issues.

This repository documents my continuous progress and practical learning throughout my Cloud Support journey.
